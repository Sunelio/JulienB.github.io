CrystalEngine

"customized Game Engine in C++."

About.

Towards the end of my second year in Game Programming studies, we undertook a substantial project. Over a four-month period, we were tasked with developing a game engine. The engine had to include essential features like animation, mesh loading, canvas editors, and a unique engine-specific feature. To enhance accessibility, we opted to script our engine in multiple languages.

Project Organization and Difficulty

Engine

We utilized OpenGL 4.6 as the foundation for developing this engine. Our task involved implementing various features, with my responsibility focusing on the implementation of canvas functionalities and animations.

My Task

Developed using the C++ language, I had to incorporate the Assimp library to parse animation information. Subsequently, I applied our approach to matrix transformations for animations, presenting a notable challenge. Another significant accomplishment was the creation of a fully editable Canvas Panel, modeled after the Canvas Panel in Unreal Engine.
